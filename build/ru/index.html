Беспроводная передача энергии имеет широкий спектр применений в потребительской электронике, автомобильной, медицинской и многих других промышленностях. Кроме того, в потребительском секторе начинает активно использоваться беспроводная передача энергии для заряда смартфонов, планшетов, носимых устройств и другой электроники.
Сегодня компания Node предлагает рынку ряд технических решений в области беспроводной передачи энергии – от потребительской электроники, до продуктов промышленного назначения.

Токены NODE, которые компания приобретет в результате обратного выкупа, будут распределены между существующими инвесторами пропорционально текущему количеству токенов на балансе на момент такого распределения.

Ежегодный обратный выкуп будет рассчитываться по следующей формуле: «Объем нераспределенной прибыли*3*Объем собранных средств/100%»

То есть в случае достижения уровня продаж в 10 млн токенов NODE, на 30% от нераспределённой прибыли будет произведен выкуп, с последующим пропорциональным распределением токенов между оставшимися инвесторами.

Таким образом, инвестор может уже сейчас рассчитать гарантированную процентную прибыль, которую он будет получать ежегодно от выкупа токенов, вне зависимости от спроса токенов NODE на рынке. Соответственно, чем дольше инвестор держит токены, тем больше дохода он сможет получить.

В любой момент времени у покупателя будет возможность приобрести продукцию NODE с 25% скидкой по отношению к стоимости в фиатной валюте (USD) при условии оплаты токенами компании. Например, при фактической стоимости Alpha 99,0USD, цена покупки при оплате токенами NODE составит 74,25 USD. Токены, которые компания приобретет в результате продажи своей продукции, будут сожжены, таким образом уменьшая их общее количество в обороте.
